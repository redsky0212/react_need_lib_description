## React 필요 library
---------------------------------

### React ( npm i react )
* React는 리액트를 사용하여 애플리케이션을 만들때 View쪽을 만들기 위한 라이브러리.
### ReactDom ( npm i react-dom )
* ReactDOM은 만들어진 UI를 실제 브라우져에 렌더링 할 때 사용하는 라이브러리.
### create-react-app으로 프로젝트 설치시 
* public/manifest.json 관련
  - 앱에 대한 정보를 담고있는 json파일 입니다.
